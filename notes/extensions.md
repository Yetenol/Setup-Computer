# Keyboard shortcuts
- `Alt + Shift + D` ‚Üê Toggle current site _# Dark Reader_
- `Ctrl + Shift + 2` ‚Üê Choose another field in KeeWeb _# KeeWeb Connect_
- `Ctrl + Shift + 1` ‚Üê One-time codes _# KeeWeb Connect_
- `Ctrl + Shift + V` ‚Üê Submit the form automatically _# KeeWeb Connect_
- `Alt + P` ‚Üê Activate the extension _# Picture-in-Picture_

# Hide from toolbar
- Click on `Extensions button` in the toolbar
- Disable `Show in toolbar` (eye icon) for all extensions
- If there is any extension left in the toolbar...
- Right-click and click `Hide from toolbar`
